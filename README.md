Python Learning Journey — Basic to Advanced

A personal roadmap and progress tracker for learning Python from scratch to an advanced level.

📌 About

This repository documents my journey learning Python, including notes, practice code, and mini-projects as I progress through each stage.


🗺️ Roadmap

1. Basics


 Installing Python & setting up the environment (VS Code / PyCharm)
 Variables & data types
 Operators (arithmetic, comparison, logical)
 Input/output (input(), print())
 Conditional statements (if, elif, else)
 Loops (for, while, break, continue)
 Strings & string methods
 Lists, tuples, sets, dictionaries
 Functions & scope
 Basic error handling (try, except)


2. Intermediate


 File handling (read/write/append)
 Modules & packages
 List/dict/set comprehensions
 Lambda functions, map, filter, reduce
 Object-Oriented Programming (classes, objects, inheritance, polymorphism)
 Exception handling (custom exceptions, finally)
 Working with JSON
 Virtual environments & pip
 Regular expressions (re module)
 Iterators & generators
